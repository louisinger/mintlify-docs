Current status:
- Closed beta on Bitcoin mainnet with select partners
- Developer SDKs available on GitHub in [Go](https://github.com/arkade-os/go-sdk), [Rust](https://github.com/arkade-os/rust-sdk), [Typescript](https://github.com/arkade-os/ts-sdk) for early builders

Roadmap:
- Q4 2025: Public mainnet launch
- Q4 2025: Expanded developer tools and additional financial primitives
- Ongoing: Multi-asset support, advanced lending protocols, and yield opportunities
