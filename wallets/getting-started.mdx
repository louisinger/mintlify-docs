# Wallet Integration Guide

Arkade integrates directly without external daemons by design. Developers can interact with the protocol using standard Bitcoin-compatible infrastructure and purpose-built native SDKs in various languages.

## **Operator Instances**

If you prefer to work with existing Arkade instances, you can connect to the following live Ark Operator instances. This allows you to test your integration against a live environment without the need for a local setup.

| **Network** | **URL**                                                           | **Status**                                                                                   |
| :---------- | :---------------------------------------------------------------- | :------------------------------------------------------------------------------------------- |
| Regtest     | [<u>http://localhost:7070</u>](http://localhost:7070)             | [<u>Self-deployed</u>](https://github.com/ArkLabsHQ/ark-docs/blob/develop/integrate/dev-env) |

## Installation

```bash
npm install @arkade/wallet-sdk
```

## Basic Setup

```typescript
import { ArkadeWallet } from '@arkade/wallet-sdk';

const identity = new InMemoryKey()
const wallet = await Wallet.create({
  identity: identity,
  arkServerUrl: 'https://mutinynet.arkade.sh',
})

// You can receive from any Arkade wallet!
const addresses = await wallet.getAddress()
console.log('Ark Address:', addresses.offchain)
```
