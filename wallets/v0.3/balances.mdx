Arkade allows you to easily check your wallet balances, both onchain and offchain. The balance information is structured to provide detailed insights into your funds.

```typescript
const balance = await wallet.getBalance()
// total is spendable in settlement
console.log('Total :', balance.total)
// available is spendable in offchain transactions
console.log('Available :', balance.available)
```

### Balance Object Structure

The balance object contains detailed information about both onchain and offchain funds:

```typescript
{
  boarding: {
    unconfirmed: 50000, // unconfirmed onchain balance in sats
    confirmed: 50000,   // confirmed onchain balance in sats
    total: 100000;      // total onchain balance (confirmed + unconfirmed) in sats
  };
  preconfirmed: 0;
  settled: 20000;         
  available: 20000;     // settled + preconfirmed
  recoverable: 5000;       // subdust and (swept=true & unspent=true)
  total: 25000;         // total offchain balance (available + recoverable) in sats
}
```

<Tip>
  **Balance Categories** / [VTXO States](/learn/pillars/vtxos#vtxo-states)

  - Onchain Balances
    - `unconfirmed`: Unconfirmed UTXO balance 
    - `confirmed`: Confirmed UTXO balance 
  - Offchain Balances
    - `preconfirmed`: Cosigned by the Arkade Signer and the sender
    - `settled`: VTXO that is Bitcoin-anchored via a [batch swap](/learn/pillars/batch-swaps)
    - `recoverable`: Offchain balance that is recoverable via an intent in a batch swap. Includes sub-dust VTXOs, expired VTXOs and Arkade Notes
</Tip>

<Tip>
  **Balance Categories** / [VTXO States](/learn/pillars/vtxos#vtxo-states)

  - `boarding.confirmed` - Confirmed boarding UTXO balance
  - `boarding.unconfirmed` - Unconfirmed boarding UTXO balance
  - `boarding.total` - Total boarding balance
  - `settled` - Settled VTXO balance
  - `preconfirmed` - Preconfirmed VTXO balance
  - `available` - Total spendable balance (settled + preconfirmed)
  - `recoverable` - Recoverable VTXO balance
  - `total` - Total balance across all categories
</Tip>

## Managing Coins

### Getting Virtual UTXOs

You can retrieve all your offchain VTXOs with a simple call:

```typescript
// Get virtual UTXOs (offchain)
const vtxos = await wallet.getVtxos()
console.log('Number of VTXOs:', vtxos.length)

// Log important information about each VTXO
vtxos.forEach((vtxo, index) => {
  console.log(`VTXO #${index + 1}:`);
  console.log(`  ID: ${vtxo.txid}:${vtxo.vout}`);
  console.log(`  Amount: ${vtxo.value} sats`);
  console.log(`  Batch ID: ${vtxo.virtualStatus.batchTxID}`);
  console.log(`  Status: ${vtxo.virtualStatus.state}`); // "preconfirmed" | "settled" | "swept" | "spent";
});
```

### Getting Boarding UTXOs

Boarding UTXOs are onchain UTXOs that have been sent to a boarding address but haven't yet been converted to VTXOs:

```typescript
// Get boarding UTXOs
const boardingUtxos = await wallet.getBoardingUtxos()
console.log('Number of boarding UTXOs:', boardingUtxos.length)

// Log important information about each boarding UTXO
boardingUtxos.forEach((utxo, index) => {
  console.log(`Boarding UTXO #${index + 1}:`);
  console.log(`  TXID: ${utxo.txid}`);
  console.log(`  Output Index: ${utxo.vout}`);
  console.log(`  Amount: ${utxo.value} sats`);
  console.log(`  Status: ${utxo.status.confirmed ? 'Confirmed' : 'Unconfirmed'}`);
});
```
