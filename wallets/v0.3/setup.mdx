## Installation

<CodeGroup>
```bash npm
npm install @arkade-os/sdk
```

```bash yarn
yarn add @arkade-os/sdk
```

```bash pnpm
pnpm add @arkade-os/sdk
```

```bash Go
go get github.com/arkade-os/go-sdk
```
</CodeGroup>

**Requirements:**
- Node.js v22+ (TypeScript/JavaScript)
- Go 1.21+ (Go)

## Create a Wallet

<CodeGroup>
```typescript TypeScript
import { Wallet, SingleKey } from '@arkade-os/sdk'

// Generate a new identity (or use an existing private key)
const identity = SingleKey.fromRandomBytes()

// Connect to an Arkade operator
const wallet = await Wallet.create({
  identity,
  arkServerUrl: 'https://mutinynet.arkade.sh'
})

// Get your receiving address
const address = await wallet.getAddress()
console.log('Address:', address)
```

```go Go
package main

import (
    "fmt"
    "log"
    
    ark "github.com/arkade-os/go-sdk"
)

func main() {
    // Generate a new identity
    identity, err := ark.NewRandomIdentity()
    if err != nil {
        log.Fatal(err)
    }

    // Connect to an Arkade operator
    wallet, err := ark.NewWallet(ark.WalletConfig{
        Identity:     identity,
        ArkServerURL: "https://mutinynet.arkade.sh",
    })
    if err != nil {
        log.Fatal(err)
    }

    // Get your receiving address
    address, err := wallet.GetAddress()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println("Address:", address)
}
```
</CodeGroup>

## Arkade Operators

Connect to a public operator or run your own:

| Network | URL | Use Case |
|---------|-----|----------|
| **Mainnet** | `https://arkade.computer` | Production |
| **Mutinynet** | `https://mutinynet.arkade.sh` | Testing (recommended) |
| **Signet** | `https://signet.arkade.sh` | Testing |
| **Regtest** | `http://localhost:7070` | Local development |

<Tip>
Use Mutinynet for development. It's a fast signet with 30-second blocks and a [faucet](https://faucet.mutinynet.com) for test funds.
</Tip>

## Using an Existing Key

If you have an existing private key (hex-encoded):

<CodeGroup>
```typescript TypeScript
import { Wallet, SingleKey } from '@arkade-os/sdk'

// Load from hex-encoded private key
const identity = SingleKey.fromHex('your-64-character-hex-private-key')

const wallet = await Wallet.create({
  identity,
  arkServerUrl: 'https://arkade.computer'
})
```

```go Go
identity, err := ark.IdentityFromHex("your-64-character-hex-private-key")
if err != nil {
    log.Fatal(err)
}

wallet, err := ark.NewWallet(ark.WalletConfig{
    Identity:     identity,
    ArkServerURL: "https://arkade.computer",
})
```
</CodeGroup>

<Warning>
Never hardcode private keys in source code. Use environment variables or a secrets manager in production.
</Warning>

## Persistent Storage

By default, wallet data is stored in memory and lost on restart. For persistent wallets, add a storage adapter:

```typescript
import { Wallet, SingleKey } from '@arkade-os/sdk'
import { FileSystemStorageAdapter } from '@arkade-os/sdk/adapters/fileSystem'

const storage = new FileSystemStorageAdapter('./wallet-data')

const wallet = await Wallet.create({
  identity: SingleKey.fromHex(process.env.PRIVATE_KEY),
  arkServerUrl: 'https://arkade.computer',
  storage  // Persists VTXOs and wallet state
})
```

See [Storage Adapters](/wallets/v0.3/storage-adapters) for browser, React Native, and other environments.

## Local Development

Run a local Arkade operator with [Nigiri](https://github.com/vulpemventures/nigiri):

```bash
# Install Nigiri
curl https://getnigiri.vulpem.com | bash

# Start with Arkade support
nigiri start --ark
```

Connect to `http://localhost:7070` for a fully isolated test environment with instant block generation.

## Verify Connection

Check that your wallet is connected:

```typescript
const balance = await wallet.getBalance()
console.log('Connected! Balance:', balance.available, 'sats')

const address = await wallet.getAddress()
console.log('Address:', address)
```
