The SDK automatically detects destination type and routes payments appropriately.

## Quick Reference

| Destination | Address Format | Speed | Fees |
|-------------|---------------|-------|------|
| **Arkade wallet** | `ark1...` / `tark1...` | Instant | Near-zero |
| **Lightning** | `lnbc...` / `lntb...` | ~30 seconds | Low (swap fee) |
| **Bitcoin onchain** | `bc1...` / `tb1...` | ~10 minutes | Network fees |

## Send to Arkade Address

<CodeGroup>
```typescript TypeScript
const txid = await wallet.sendBitcoin({
  address: 'ark1q...',
  amount: 50000  // sats
})

console.log('Sent:', txid)
```

```go Go
txid, err := wallet.SendBitcoin(SendBitcoinParams{
    Address: "ark1q...",
    Amount:  50000,
})
if err != nil {
    log.Fatal(err)
}

fmt.Println("Sent:", txid)
```
</CodeGroup>

Payments to Arkade addresses are instant and cost fractions of a satoshi. The recipient doesn't need to be online.

## Send to Lightning Invoice

<CodeGroup>
```typescript TypeScript
const txid = await wallet.sendBitcoin({
  address: 'lnbc50u1p...',  // Lightning invoice
  amount: 50000
})
```

```go Go
txid, err := wallet.SendBitcoin(SendBitcoinParams{
    Address: "lnbc50u1p...",
    Amount:  50000,
})
```
</CodeGroup>

Lightning payments use [submarine swaps](/contracts/lightning-swaps) under the hood. The SDK handles swap coordination automatically.

<Info>
For advanced Lightning integration (generating invoices, custom swap handling), see the [Lightning Swaps guide](/contracts/lightning-swaps).
</Info>

## Send to Bitcoin Address

<CodeGroup>
```typescript TypeScript
const txid = await wallet.sendBitcoin({
  address: 'bc1q...',  // Onchain Bitcoin address
  amount: 100000
})
```

```go Go
txid, err := wallet.SendBitcoin(SendBitcoinParams{
    Address: "bc1q...",
    Amount:  100000,
})
```
</CodeGroup>

Onchain sends trigger a [collaborative exit](/wallets/v0.3/ramps#exiting) that settles in the next batch. Standard Bitcoin network fees apply.

<Tip>
For larger withdrawals, use `wallet.offboard()` directly for more control over timing and fees.
</Tip>

## Fee Comparison

| Destination | Typical Fee | Notes |
|-------------|-------------|-------|
| Arkade | <1 sat | Offchain execution fee |
| Lightning | ~0.1% | Submarine swap + routing |
| Bitcoin | 500-5000 sats | Depends on network congestion |

## Error Handling

```typescript
try {
  const txid = await wallet.sendBitcoin({
    address: recipientAddress,
    amount: 50000
  })
  console.log('Success:', txid)
} catch (error) {
  if (error.code === 'INSUFFICIENT_BALANCE') {
    console.error('Not enough funds')
  } else if (error.code === 'INVALID_ADDRESS') {
    console.error('Unrecognized address format')
  } else if (error.code === 'OPERATOR_OFFLINE') {
    console.error('Operator unavailable - try again later')
  } else {
    console.error('Send failed:', error.message)
  }
}
```

## Checking Balance Before Send

```typescript
const balance = await wallet.getBalance()

const amount = 50000
if (balance.available < amount) {
  console.error(`Insufficient funds: ${balance.available} < ${amount}`)
  return
}

await wallet.sendBitcoin({ address, amount })
```
