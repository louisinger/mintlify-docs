Create a wallet, generate an address, and send an instant Bitcoin payment. This guide uses the TypeScript SDK on Mutinynet (testnet).

<Tip>
Want a complete end-to-end walkthrough? Check out [Zero to One](/start/zero-to-one) — it covers boarding, sending, settling, and exiting in 15 minutes.
</Tip>

## Prerequisites

- Node.js v22 or higher
- npm or yarn

## 1. Install the SDK

```bash
mkdir arkade-quickstart && cd arkade-quickstart
npm init -y
npm install @arkade-os/sdk
```

## 2. Create a Wallet

Create `wallet.ts`:

```typescript
import { Wallet, SingleKey } from '@arkade-os/sdk'
import { randomBytes } from 'crypto'

async function main() {
  // Generate a random private key (use secure storage in production)
  const privateKey = randomBytes(32).toString('hex')
  console.log('Private key (save this):', privateKey)
  
  // Create wallet identity from the private key
  const identity = SingleKey.fromHex(privateKey)
  
  // Connect to Mutinynet testnet
  const wallet = await Wallet.create({
    identity,
    arkServerUrl: 'https://mutinynet.arkade.sh'
  })
  
  // Get your Arkade address for receiving payments
  const address = await wallet.getAddress()
  console.log('Arkade address:', address)
  
  // Check your balance
  const balance = await wallet.getBalance()
  console.log('Balance:', balance.available, 'sats')
}

main().catch(console.error)
```

Run it:

```bash
npx ts-node wallet.ts
```

Output:

```
Private key (save this): a1b2c3d4...
Arkade address: tark1q...
Balance: 0 sats
```

<Warning>
Save your private key. You'll need it to access your wallet again.
</Warning>

## 3. Get Test Funds

Arkade addresses (`tark1...`) receive funds from other Arkade wallets or via [Lightning swaps](/contracts/lightning-swaps). For testing, you have two options:

**Option A: Receive from another Arkade user**

Share your `tark1...` address with someone who already has testnet funds.

**Option B: Onboard via boarding address**

Generate a standard Bitcoin address and fund it from a [Mutinynet faucet](https://faucet.mutinynet.com):

```typescript
// Get a boarding address (standard Bitcoin address)
const boardingAddress = await wallet.getBoardingAddress()
console.log('Boarding address:', boardingAddress)
// Fund this from faucet.mutinynet.com, then run:
// await new Ramps(wallet).onboard()
```

See [Ramps](/wallets/v0.3/ramps) for the complete onboarding process.

## 4. Send a Payment

Once funded, sending is instant:

```typescript
import { Wallet, SingleKey } from '@arkade-os/sdk'

async function send() {
  // Use your saved private key
  const identity = SingleKey.fromHex('your-saved-private-key')
  
  const wallet = await Wallet.create({
    identity,
    arkServerUrl: 'https://mutinynet.arkade.sh'
  })
  
  // Check balance
  const balance = await wallet.getBalance()
  console.log('Balance:', balance.available, 'sats')
  
  // Send to another Arkade address
  const txid = await wallet.sendBitcoin({
    address: 'tark1q...recipient-address...',
    amount: 1000  // satoshis
  })
  
  console.log('Sent! Transaction ID:', txid)
}

send().catch(console.error)
```

The payment confirms instantly—no waiting for blocks.

## What You Built

You created a self-custodial Bitcoin wallet that:

- **Sends instant payments** via [VTXOs](/learn/pillars/vtxos) in the [virtual mempool](/learn/pillars/virtual-mempool)
- **Maintains full control** with presigned exit transactions to Bitcoin
- **Pays near-zero fees** for offchain transfers
